# Bhone's Personal Portfolio Website

A fully responsive personal portfolio website built with Bootstrap 4, customized from the Start Bootstrap Agency template. The site showcases my projects, skills, academic background, and personal interests.

## Live Sections

| # | Section | Description |
|---|---------|-------------|
| 1 | **About Me** | Personal bio, location, field of study, and interests |
| 2 | **Skills** | Front-end, back-end, and responsive design expertise with animated progress bars |
| 3 | **Projects** | Real project showcases with detail modals |
| 4 | **Academics** | Timeline of my educational journey from high school to current internship |
| 5 | **Favourites** | Favourite activities, food, and sports |
| 6 | **Contact** | Contact form for reaching out |

## Featured Projects

### Restaurant Management App
A comprehensive full-stack web application for automating restaurant operations.
- **Frontend:** React 18, React Router v6, Context API, Socket.IO Client
- **Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, bcryptjs
- **Features:** Role-based dashboards (Admin/Waiter/Chef), real-time order tracking, billing with partial payments, sales analytics, printable bills

### Restaurant Chatbot
An NLP-powered chatbot for restaurant ordering and assistance.
- **Technologies:** Python, Scikit-learn, NLTK, Sentence-Transformers, PyTorch
- **Features:** Intent classification trained on CLINC150 + custom intents, Named Entity Recognition for menu items/quantities, multi-turn conversation state management

## Tech Stack

## How to View

Simply open `index.html` in any modern web browser. No build tools or server required.

## Responsive Design

The website is fully responsive and adapts to all screen sizes:
- **Desktop** (1200px+) - Full layout with side-by-side content
- **Tablet** (768px - 1199px) - Adjusted grid and spacing
- **Mobile** (< 768px) - Stacked layout with collapsible navigation
