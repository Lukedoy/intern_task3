Restaurant Management App

A comprehensive full-stack system to automate restaurant operations.

A full-stack web application designed to streamline restaurant workflows. It features role-based dashboards for Admin, Waiter, and Chef roles. Waiters manage tables with a split-panel view and place orders inline. Chefs receive a real-time order queue with per-item status tracking (confirmed, preparing, ready). Admins oversee menu management, user accounts, table configuration, and sales analytics.

Key features include real-time updates via Socket.IO, a billing system with discount support, 5% tax calculation, and partial payment tracking (paid/unpaid items). The app also supports takeaway orders, order history with search/filter/sort/pagination, and printable formatted bills.

  • Frontend: React 18, React Router v6, Context API, Socket.IO Client
  • Backend: Node.js, Express.js, MongoDB, Mongoose, JWT, bcryptjs
  • Testing: Jest, React Testing Library, Supertest
  • Version: 2.0.0
  • Status: Completed

Restaurant Chatbot

An NLP-powered chatbot for restaurant ordering and assistance.

Built an intelligent chatbot using Natural Language Processing (NLP) techniques trained on the CLINC150 dataset combined with custom restaurant-specific intents. The system trains and compares three ML models, automatically selecting the best performer for deployment.

The chatbot understands intents like greetings, menu inquiries, ordering items, price questions, and recommendations. It uses a custom Named Entity Recognition (NER) module to extract menu items and quantities from user input, maintains conversational state for multi-turn interactions, and supports contextual references (e.g., "it", "that") by tracking the last mentioned item.

  • Technologies: Python, Scikit-learn, NLTK, Sentence-Transformers, PyTorch
  • NLP: Intent classification, NER, text cleaning, TF-IDF vectorization
  • Dataset: CLINC150 + Custom restaurant intents
  • Architecture: Modular (bot logic, NLP pipeline, training, evaluation)
  • Status: Completed (v1.1)
